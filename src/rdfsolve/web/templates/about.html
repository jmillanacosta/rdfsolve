About RDFSolve

What is RDFSolve?

RDFSolve is a comprehensive tool for analyzing RDF (Resource Description Framework) datasets. It provides functionality to extract schemas, generate VoID descriptions, and perform detailed analysis of RDF graphs accessible through SPARQL endpoints.

Features
  • Schema Extraction: Automatically extract and analyze RDF schemas from SPARQL endpoints
  • VoID Generation: Generate comprehensive VoID (Vocabulary of Interlinked Datasets) descriptions
  • Dataset Analysis: Perform statistical analysis of RDF datasets including class and property counts
  • Multiple Formats: Export results in various formats including CSV, JSON, and Turtle
  • Command Line Interface: Powerful CLI tools for batch processing and automation
  • Web Interface: User-friendly web interface for interactive exploration
Getting Started

RDFSolve can be used both as a Python library and through its command-line interface. The tool supports analysis of various RDF datasets including UniProt, ChEMBL, DrugBank, PubChem, and many others.

Check out the notebooks in the notebooks/ directory for examples of how to analyze specific datasets.
